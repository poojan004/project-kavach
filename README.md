# Project KAVACH
AI-Enabled Cybersecurity Engagement Project
IIT Roorkee × Futurense AI Clinic

## Overview

Project KAVACH is a combined cybersecurity engagement focused on:

- Network Forensics
- Web Application Security
- Threat Modeling
- Defense in Depth
- Secure Remediation

The project simulates a security assessment for a fictional NBFC organization called Meridian FinServe.

The engagement includes:
- PCAP analysis
- IOC extraction
- Vulnerability assessment
- OWASP Top 10 exploitation
- Secure remediation
- SAST validation
- Executive security reporting

## Workstreams

### Workstream A — Network Forensics
- PCAP triage
- IOC extraction
- Threat hypothesis analysis
- Network architecture hardening

### Workstream B — Web Application Security
- DVWA assessment
- OWASP Juice Shop testing
- OWASP Top 10 validation
- Secure remediation

### Workstream C — Synthesis
- Joint threat model
- Defense-in-depth proposal
- Executive readout
